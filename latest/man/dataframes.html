    Reading into DataFrames

    Reading into DataFrames

    uCSV implements a convenience constructor for DataFrames that takes the output of uCSV.read (a Tuple{Vector::Any, Vector{String}}) and converts it to a DataFrame.

    Note

    When NamedTuples becomes available in a release version of Julia, uCSV read will return the data as a NamedTuple object and this function will be deprecated (since DataFrames will implement its own constructor for NamedTuples)

    julia> using uCSV, DataFrames

julia> s =
       """
       1.0,1.0,1.0
       2.0,2.0,2.0
       3.0,3.0,3.0
       """;

julia> DataFrame(uCSV.read(IOBuffer(s)))
3×3 DataFrames.DataFrame
│ Row │ x1  │ x2  │ x3  │
├─────┼─────┼─────┼─────┤
│ 1   │ 1.0 │ 1.0 │ 1.0 │
│ 2   │ 2.0 │ 2.0 │ 2.0 │
│ 3   │ 3.0 │ 3.0 │ 3.0 │

    And again, but with a header

    julia> using uCSV, DataFrames

julia> s =
       """
       c1,c2,c3
       1,1.0,a
       2,2.0,b
       3,3.0,c
       """;

julia> DataFrame(uCSV.read(IOBuffer(s), header = 1))
3×3 DataFrames.DataFrame
│ Row │ c1 │ c2  │ c3 │
├─────┼────┼─────┼────┤
│ 1   │ 1  │ 1.0 │ a  │
│ 2   │ 2  │ 2.0 │ b  │
│ 3   │ 3  │ 3.0 │ c  │
